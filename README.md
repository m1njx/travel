# ✈️ TripSync (유럽 여행 일정 관리 및 실시간 동기화 플래너)

TripSync는 팀원들과 실시간으로 유럽 여행 일정을 계획하고, 지출 내역을 기록하며, 복잡한 경비를 간편하게 정산할 수 있도록 돕는 실시간 협업형 스마트 웹 애플리케이션입니다. Google Gemini AI 비서 기능과 Firebase 실시간 동기화를 바탕으로 최적화된 여행 환경을 선사합니다.

---

## 🚀 주요 기능 상세 소개

### 1. 👥 실시간 방(Room) & 팀원 관리
* **권한별 역할 분리**: 방을 개설한 **관리자(Admin)**와 초대 코드로 접속하는 **멤버(Member)** 역할로 구분됩니다.
* **멤버 초대 코드(6자리 CODE) 생성**:
  * 관리자는 멤버를 추가할 때 고유한 **6자리 영문/숫자 조합의 초대 코드**를 지정하거나 생성합니다.
  * 일반 멤버는 패스워드 없이 이 초대 코드 입력만으로 본인의 프로필에 매핑되어 간편하게 로그인할 수 있습니다.
* **소그룹 팀 매핑(Team Mapping)**:
  * 전체 인원 중 특정 멤버들을 소그룹 팀으로 묶을 수 있습니다.
  * 일정 작성 시 특정 팀을 지정하면, **해당 팀에 속한 멤버들에게만 일정이 필터링되어 노출**되는 맞춤형 뷰를 제공합니다.

### 2. 📍 여행 일정 플래너 & AI 동선 최적화
* **인터랙티브 타임라인**: 날짜별 방문지의 시간표와 세부 동선 메모를 실시간으로 조율합니다.
* **구글 지도 연동**: 방문지의 위치명을 원클릭으로 구글 지도(Google Maps) 검색 페이지와 연결하여 즉시 위치를 확인할 수 있습니다.
* **Gemini AI 동선 최적화 (AI Route Optimize)**:
  * 현재 등록된 일정 장소들의 지리적 근접성을 AI가 분석하여 **가장 효율적인 동선 순서로 재배열**합니다.
  * 현실적인 이동 시간을 감안하여 일정이 자동으로 조율되며, 방문 장소별 꿀팁(예: 뮤지엄패스 필수, 예약 팁 등)을 담은 한글 메모를 채워줍니다.
  * 주변에 추천할 만한 핵심 관광 명소가 있다면 AI가 판단하여 신규 일정 후보로 직접 추천해 줍니다.

### 3. 💶 가계부 & 정밀 N분의 1 정산 (Dutch Pay)
* **AI 영수증 스캔 (AI Receipt Scanner)**:
  * 영수증 사진(카메라 촬영 또는 갤러리 업로드)을 올리면 Gemini AI가 **금액, 통화(EUR, USD 등), 매장명(한글 번역), 지출 카테고리**를 자동으로 파악하여 폼에 자동 입력해 줍니다.
* **다국어 통화 지원**: 유럽 국가별 특성을 고려하여 EUR, GBP, CHF, USD, CZK, HUF, PLN, SEK, NOK, DKK, KRW 등 다양한 통화 입력을 지원합니다.
* **실시간 환율 수집**: Gemini Search Grounding 및 백업 공시 API를 활용해 오늘의 최신 환율 정보를 자동으로 불러옵니다.
* **정밀 네덜란드 페이(Dutch Pay) 계산기**:
  * 지출 등록 시 참여 멤버(누가 같이 썼는지)를 개별적으로 선택할 수 있습니다.
  * 총지출과 각 멤버별 지출 내역을 종합하여 **최종적으로 '누가 누구에게 얼마를 송금해야 하는지' 최적의 정산 경로**를 자동으로 도출합니다.

### 4. 📊 유럽 전용 가계부 (Euro Expense Manager)
* **도시별 경비 분류**: 파리, 로마, 런던 등 여행 도시별 지출 내역을 별도 카드로 나누어 직관적으로 통제합니다.
* **결제 수단 필터링**: 현금, 카드뿐만 아니라 유럽 여행 필수품인 **트래블로그, 트래블월렛** 등 결제 수단별 통계를 제공합니다.
* **대시보드 통계**: 전체 지출 대비 카테고리별 비중(식비, 숙박, 교통, 쇼핑 등) 및 도시별 예산 대비 지출 비중을 시각적인 차트와 함께 요약해서 보여줍니다.

### 5. 📋 체크리스트 & AI 준비물 추천
* **체크리스트 분류**: 개인 준비물과 팀 공용 준비물을 구분하여 꼼꼼하게 진행 상황을 체크합니다.
* **AI 기반 맞춤형 패킹 가이드**:
  * 가고자 하는 여행지와 날씨/시기 정보를 입력하면, Gemini AI가 현지 수질, 치안, 기후 환경에 꼭 필요한 **최적의 준비물 리스트(5~8개)와 구체적인 추천 사유**를 자동 생성합니다.
  * 마음에 드는 추천 아이템들을 클릭 한 번으로 내 체크리스트에 일괄 등록할 수 있습니다.

### 6. 🌤️ 날씨 위젯 & AI 실시간 가이드
* **실시간 날씨 예보**: 사용자의 GPS 좌표를 기반으로 OpenStreetMap Nominatim 역지오코딩을 적용해 현재 머무는 도시 이름을 표시하며, Open-Meteo API를 연동하여 현재 날씨 및 향후 24시간 동안의 시간별 온도/강수확률 예보를 제공합니다.
* **AI 맞춤 여행 꿀팁(Daily AI Tip)**:
  * 현재 남은 일정 개수, 준비물 완성도, 오늘 지출 규모를 AI가 실시간으로 종합 분석하여 **유용한 당일치기 팁(환전/소매치기 예방 등)을 2~3줄의 다정한 말투로 가이드**해 줍니다.

### 7. 📝 실시간 공유 메모
* 포스트잇 형태의 감각적인 보드에서 팀원들이 자유롭게 정보를 작성하고 실시간 동기화합니다.
* 중요한 메모는 **상단 핀(Pin)** 기능을 통해 고정할 수 있으며, 카테고리별로 모아보기가 가능합니다.

### 8. 🪵 활동 내역 로그 (Activity Log)
* 어떤 멤버가 어떤 데이터(일정 추가, 예산 삭제, 체크리스트 달성 등)를 언제 변경했는지 모든 이력을 타임라인 형태로 명확히 기록하여 협업 시 혼선을 예방합니다.

## 🔒 보안 및 API 호출 최적화 조치

1. **Gemini API 호출 제한(429) 방지용 10분 캐싱 알고리즘**
   * Gemini API Free Tier의 분당 15회 호출 제한에 대응하여, 한번 조회된 실시간 환율 정보는 `localStorage`에 캐싱됩니다.
   * [ExpensePage](file:///Users/kangminje04/travel/src/pages/ExpensePage.jsx)가 마운트될 때마다 API를 새로 호출하지 않고 **10분의 유효시간** 동안 캐시 데이터를 먼저 참조합니다.
   * React Strict Mode(개발 환경)로 인한 마운트 즉시 연속 이중 API 요청 문제 역시 캐시 선조회를 통해 완벽히 해결되었습니다.
2. **소스코드 보안**
   * Firebase Config 및 API Key 등 모든 중요 크리덴셜은 환경 변수 파일(`.env`)에서만 제어되며, Git 추적에서 철저히 제외됩니다.
3. **리버스 탭내빙(Reverse Tabnabbing) 취약점 제어**
   * 구글 맵 등으로 열리는 모든 외부 새 창(`target="_blank"`)에 `rel="noopener noreferrer"` 보안 속성을 주입하여 부모 창 탈취 위협을 원천 차단했습니다.
4. **종속성 취약점 패치 완료**
   * 패키지 감사(`npm audit`)를 통해 발견된 모든 종속성 라이브러리의 보안 경고를 `npm audit fix`를 통해 조치하여 **보안 취약점 0개** 클린 상태를 보증합니다.

## 🚀 실행 및 로컬 설치 방법

### 1. 사전 요구 사양
- Node.js >= 20.0.0

### 2. `.env` 파일 환경 변수 구성
Vite 빌드 사양에 맞추어 프로젝트 루트 폴더에 `.env` 파일을 생성하고 아래 키를 입력해 주세요.

```env
VITE_GEMINI_API_KEY=your_gemini_api_key
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.firebasestorage.app
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 3. 프로젝트 구동 명령어
```bash
# 1. 의존성 모듈 설치
npm install

# 2. 로컬 개발 환경 서버 구동
npm run dev

# 3. 배포(Render 등) 빌드 검증
npm run build
```
