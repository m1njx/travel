# ✈️ TripSync (유럽 여행 일정 관리 및 실시간 동기화 플래너)

TripSync는 팀원들과 실시간으로 유럽 여행 일정을 계획하고, 지출 내역을 기록하며, 복잡한 경비를 간편하게 정산할 수 있도록 돕는 실시간 협업형 스마트 웹 애플리케이션입니다. Google Gemini AI 비서 기능과 Firebase 실시간 동기화를 바탕으로 최적화된 여행 환경을 선사합니다.

---

## 🚀 주요 기능 상세 소개

### 1. 👥 실시간 방(Room) & 멤버 관리
* **권한별 역할 분리**: 방을 최초 개설한 **관리자(Admin)**와 초대 코드로 접속하는 **일반 멤버(Member)** 역할로 구분되어 데이터를 안전하게 제어합니다.
* **멤버 초대 코드(6자리 CODE) 생성**:
  * 관리자는 신규 멤버를 추가할 때 고유한 **6자리 영문/숫자 조합의 초대 코드**를 지정하거나 랜덤 생성하여 부여합니다.
  * 멤버는 별도의 복잡한 패스워드 설정 없이, 부여받은 초대 코드를 입력하여 로그인하면 자신의 프로필로 즉시 매핑됩니다.
* **소그룹 팀 매핑(Team Mapping)**:
  * 전체 인원 중 특정 멤버들을 소그룹(예: 1팀, A조 등)으로 묶을 수 있습니다.
  * 일정 작성 시 특정 팀을 타겟팅하여 지정하면, **해당 팀에 속한 멤버들에게만 일정이 필터링되어 노출**되는 맞춤형 뷰를 제공합니다.

### 2. 📍 여행 일정 플래너 & AI 동선 최적화
* **인터랙티브 타임라인**: 날짜별 방문지의 시간표와 세부 동선 메모를 작성하고 실시간 동기화합니다.
* **구글 지도 연동**: 등록된 장소명을 클릭하면 새 창으로 구글 지도(Google Maps) 검색 페이지와 즉시 연동되어 위치 파악이 편리합니다.
* **Gemini AI 동선 최적화 (AI Route Optimize)**:
  * 현재 등록된 일정 장소들의 지리적 근접성을 AI가 분석하여 **가장 효율적인 이동 경로 순서로 재배열**합니다.
  * 현실적인 이동 시간 및 교통 수단을 예측하여 일정을 자동으로 재구성하고, 현지 방문 꿀팁(예: 뮤지엄패스 활용, 예약 대기 팁 등)을 담은 상세 한글 메모를 채워줍니다.
  * 주변의 유망한 관광지나 숨겨진 맛집 후보를 분석하여 추가 일정으로 직접 추천해 줍니다.

### 3. 💶 가계부 & 정밀 N분의 1 정산 (Dutch Pay)
* **AI 영수증 스캔 (AI Receipt Scanner)**:
  * 현지에서 받은 영수증 사진(카메라 촬영 또는 갤러리 이미지)을 업로드하면 Gemini AI가 **금액, 통화(EUR, USD 등), 가맹점명(한글 번역 및 정제), 지출 카테고리**를 파악해 폼에 자동으로 채워넣습니다.
* **다국어 통화 지원**: 유럽 다국가 이동 경로를 고려하여 EUR, GBP, CHF, USD, CZK, HUF, PLN, SEK, NOK, DKK, KRW 등 다양한 현지 통화 입력을 지원합니다.
* **실시간 환율 수집 및 10분 캐싱**:
  * Gemini Search Grounding 및 백업 환율 공시 API를 활용해 매일 실시간 환율 정보를 자동으로 업데이트합니다.
  * 잦은 호출로 인한 429(Rate Limit) 예외를 방지하고자 **10분간 유효한 로컬 캐싱 알고리즘**을 내장하였습니다.
* **정밀 네덜란드 페이(Dutch Pay) 최적화**:
  * 지출을 등록할 때 돈을 함께 쓴 참여 멤버를 개별 체크하여 기록할 수 있습니다.
  * 모든 지출 내역과 참여 지분을 계산하여 **'누가 누구에게 얼마를 송금해야 하는지' 최적의 송금 경로(송금 횟수 최소화)**를 산출합니다.

### 4. 📊 유럽 전용 가계부 (Euro Expense Manager)
* **도시별 경비 분할 카드**: 파리, 로마, 런던, 인터라켄 등 방문하는 도시별 지출 내역을 직관적인 별도 카드 섹션으로 분할하여 통제합니다.
* **결제 수단 기본값 개선 & 통계**: 
  * 현금, 카드뿐만 아니라 유럽 여행 필수 카드인 **트래블로그, 트래블월렛** 등 결제 수단별 상세 내역을 필터링하여 확인합니다.
  * 지출 등록 시 여행 트렌드를 반영해 **기본 선택 결제 수단이 '카드'로 지정**되어 있어 등록 효율성을 높였습니다.
* **대시보드 통계**: 전체 지출 대비 카테고리별 비중(식비, 숙박, 교통, 쇼핑, 티켓 등) 및 각 도시별 예산 대비 누적 지출 비중을 미려한 반응형 차트로 시각화합니다.

### 5. 📋 체크리스트 & AI 준비물 추천
* **개인/공용 항목 분류**: 개인 소지품과 팀원 전체가 공동으로 준비해야 할 항목(멀티탭, 상비약 등)을 구분하여 관리합니다.
* **AI 기반 맞춤형 패킹 가이드**:
  * 여행할 국가/도시와 해당 시기 정보를 입력하면 Gemini AI가 현지 수질, 치안, 기후 환경을 고려하여 꼭 필요한 **최적의 준비물 리스트(5~8개)와 필수 추천 사유**를 자동 브리핑합니다.
  * 추천 항목을 클릭 한 번으로 내 체크리스트에 빠르게 추가 등록할 수 있습니다.

### 6. 🌤️ 날씨 위젯 & AI 실시간 가이드
* **실시간 날씨 예보**: 사용자의 GPS 좌표를 기반으로 OpenStreetMap Nominatim 역지오코딩을 적용해 머물고 있는 도시의 한글 명칭을 표시하며, Open-Meteo API를 연동하여 시간별 온도 및 강수확률 예보를 미려한 애니메이션 아이콘과 함께 제공합니다.
* **AI 맞춤 여행 꿀팁(Daily AI Tip)**:
  * 대시보드에서 남은 일정 현황, 체크리스트 달성도, 오늘 지출 규모를 AI가 실시간으로 종합 분석하여 **유용한 하루 가이드(환전 타이밍, 야간 소매치기 예방 요령 등)를 다정하게 브리핑**합니다.

### 7. 🚨 대사관 긴급 SOS 위젯 (New)
* **사이렌 긴급 팝업**: 대시보드 상단에 배치된 pulsing 애니메이션 효과의 **빨간색 사이렌 버튼**을 터치하면 토스 스타일의 긴급 SOS 모달 시트가 호출됩니다.
* **11개 주요 유럽 도시 연락처 매핑**:
  * **대상 도시**: 런던, 파리, 인터라켄, 뮌헨, 프라하, 비엔나, 부다페스트, 베니스, 피렌체, 로마, 바르셀로나
  * 각 도시의 대한민국 대사관 및 총영사관, 그리고 외교부 영사콜센터(24h)의 **실제 가용한 대표 전화 및 사건·사고 당직 연락처**를 완벽하게 검증하여 탑재했습니다.
  * **원터치 긴급 통화**: 모바일 기기 터치 시 즉시 전화 통화 화면(`tel:` 프로토콜 연동)으로 이동되어 위급 상황 발생 시 극도의 신속함을 보장합니다.

### 8. 🧳 유럽 여행 도구 & 교통 꿀팁 (New)
* **한국-현지 실시간 시차 비교**: 한국 시간(KST)과 내가 위치한 영국(BST) 또는 유럽 대륙(CEST)의 현지 시간을 나란히 보여주며, 한국에 있는 가족이나 친구에게 안부 연락을 취하기 가장 좋은 시간대인지 직관적인 안부 조언 알림(전화하기 좋은 시간, 취침 시간 등)을 표시합니다.
* **유럽 택스 리펀 (Tax Refund) 계산기 & 가이드**: 프랑스, 이탈리아, 스페인, 독일 등 유럽 각국별 최소 구매 금액 한도와 고유 환급 비율을 바탕으로 예상 세금 환급액을 즉석에서 모의 계산하고, 최종 출국 시 공항 세관(Customs)에서 텍스 리펀을 성공적으로 완료하기 위해 놓쳐서는 안 될 필수 행동 요령 체크리스트를 상세히 제공합니다.
* **대중교통 꿀팁 및 벌금 예방 가이드**: 런던 비접촉 신용카드 사용 팁부터 파리 지하철 종이 티켓 출구 전 보관 의무, 그리고 이탈리아/독일/체코/오스트리아 기차 및 대중교통 탑승 전 승차권 각인(펀칭/Check-in)을 누락하여 억울하게 수십 유로 이상의 무거운 벌금을 물게 되는 상황을 완벽하게 방지하기 위한 각 나라별 맞춤 팁을 요약하여 알려줍니다.

### 9. 📝 실시간 공유 메모
* 포스트잇 형태의 감각적인 메모 보드로 팀원들이 자유롭게 텍스트 및 정보를 기록하고 실시간 동기화합니다.
* 중요 메모는 **상단 핀(Pin)** 기능을 통해 고정할 수 있으며, 카테고리 필터링이 가능합니다.

### 10. 🪵 활동 내역 로그 (Activity Log)
* 어떤 멤버가 어떤 데이터(일정 추가, 예산 삭제, 체크리스트 달성 등)를 언제 변경했는지 모든 이력을 타임라인 형태로 명확히 기록하여 협업 시 발생할 수 있는 혼선을 미연에 방지합니다.

## 🔒 보안 및 API 호출 최적화 조치

1. **Gemini API 호출 제한(429) 방지용 10분 캐싱 알고리즘**
   * Gemini API Free Tier의 분당 15회 호출 제한에 대응하여, 한번 조회된 실시간 환율 정보는 `localStorage`에 캐싱됩니다.
   * [ExpensePage](file:///Users/kangminje04/travel/src/pages/ExpensePage.jsx)가 마운트될 때마다 API를 새로 호출하지 않고 **10분의 유효시간** 동안 캐시 데이터를 먼저 참조합니다.
   * React Strict Mode(개발 환경)로 인한 마운트 즉시 연속 이중 API 요청 문제 역시 캐시 선조회를 통해 완벽히 해결되었습니다.
2. **소스코드 보안**
   * Firebase Config 및 API Key 등 모든 중요 크리덴셜은 환경 변수 파일(`.env`)에서만 제어되며, Git 추적에서 철저히 제외됩니다.
3. **리버스 탭내빙(Reverse Tabnabbing) 취약점 제어**
   * 구글 맵 등으로 열리는 모든 외부 새 창(`target="_blank"`)에 `rel="noopener noreferrer"` 보안 속성을 주입하여 부모 창 탈취 위협을 원천 차단했습니다.
4. **종속성 취약점 패치 완료**
   * 패키지 감사(`npm audit`)를 통해 발견된 모든 종속성 라이브러리의 보안 경고를 `npm audit fix`를 통해 조치하여 **보안 취약점 0개** 클린 상태를 보증합니다.

## 🚀 실행 및 로컬 설치 방법

### 1. 사전 요구 사양
- Node.js >= 20.0.0

### 2. `.env` 파일 환경 변수 구성
Vite 빌드 사양에 맞추어 프로젝트 루트 폴더에 `.env` 파일을 생성하고 아래 키를 입력해 주세요.

```env
VITE_GEMINI_API_KEY=your_gemini_api_key
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.firebasestorage.app
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 3. 프로젝트 구동 명령어
```bash
# 1. 의존성 모듈 설치
npm install

# 2. 로컬 개발 환경 서버 구동
npm run dev

# 3. 배포(Render 등) 빌드 검증
npm run build
```
